
Write the Binary-Search-Tree stages of the sequence [7, 5, 1, 8, 3, 6, 0, 9, 4, 2].
             

#Solution:

```

         5    -> root
     /       \ 
    2         7  -> nodes
   / \       / \
  1   3     6   8  -> nodes
 /     \         \ 
0       4         9  -> nodes

```
